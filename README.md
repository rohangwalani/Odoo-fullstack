# Hackathon Project Repository

## Project Structure
- `/backend`: Spring Boot Application
- `/frontend`: React Application

## How to get started
1. Make sure you are on the `main` branch and run `git pull origin main` to get the latest updates.
2. Create your own feature branch before coding: `git checkout -b yourname-feature`
3. Commit often using conventions like `[FEAT]`, `[FIX]`, `[UI]`.
4. Open a Pull Request on GitHub to merge into `main`. Do not push directly to `main`!

## Backend setup
- Update `application.properties` with your local MySQL credentials.
- Import the `api-collection.json` into Postman to test endpoints locally.
